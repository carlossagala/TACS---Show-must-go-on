TACS 1C2017 - TP grupal

El objetivo del TP es desarrollar una aplicación que permita a los usuarios interactuar con la API de The Movie Database creando así una app con diversos casos de uso relacionados a las películas y actores provistos por la API.
La aplicación funcionará de modo stand alone, y estará publicada en la nube para ser accedida.
